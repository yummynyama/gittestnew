Есть два быстрых способа сделать ER Диаграмму.

Первый способ, если есть есть DBeaver, то:

1. Вкладка Database -> New Database Conncetion ctrl+shift+N -> SQLite -> Во вкладке General будет пункт Path: и нужно будет нажать на кнопку Open для выбора .SQLite файла БД и нажимаем на кнопку Done

2. Нажимаем на БД -> Tables -> далее ПКМ по Tables и выбрать пункт View Diagram.

3. Делаем скриншот и готова.


Второй способ, если нету DBeaver, то:

Есть скрипт, который нужно запустить db_to_mermaid через команду python db_to_mermaid и готова, скрипт сам найдет бдшку и сделает файлик, который нужно будет открыть в vs code, тока для vscode нужно будет установить плагин Mermaid
